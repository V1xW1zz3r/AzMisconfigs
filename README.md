# AzMisconfigs
A set of collection of misconfigured azure stuff like ARM templates or so
You can easily spin up an poorly configured enivornment for your own testing, tools and whatnot, and feel free to configure these to meet your needs

## Inspiration
I made this for testing purposes, inspired by the book: [Penetration Testing Azure](https://www.amazon.com/Penetration-Testing-Azure-Ethical-Hackers/dp/1839212934)
Some of the config files are obsolete, which won't work, so I made a morden version for it
